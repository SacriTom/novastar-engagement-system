## Design Standards (All Templates)

| Element | Specification |
|---------|---------------|
| From name | "NovaStar [City]" (not "NovaStar Hotels Marketing") |
| Reply-to | Monitored inbox (not no-reply) |
| Unsubscribe | One-click, top and bottom of email |
| Mobile responsive | Required; single-column layout, minimum 44px tap targets |
| Image-to-text ratio | Max 40% image, 60% text |
| Preheader | Always populated; never duplicate the subject line |
| Personalisation fallback | If first name unavailable, use "Hi there" (never "Dear Guest") |
